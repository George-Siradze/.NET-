# Loan Management System API

ეს არის მარტივი სასესხო სისტემის სერვერული ნაწილი (REST API), რომელიც აგებულია ASP.NET Core-ზე. სისტემა საშუალებას გაძლევთ მართოთ მომხმარებლები, ავტომატიზებულად გასცეთ სესხები და აღრიცხოთ გადახდები.

## გამოყენებული ტექნოლოგიები
* **ჩარჩო (Framework):** .NET 8 (ASP.NET Core Web API)
* **ORM:** Entity Framework Core
* **მონაცემთა ბაზა:** SQL Server
* **დოკუმენტაცია:** Swagger

## ძირითადი ფუნქციონალი
* **სესხის დამტკიცება/უარყოფა:** სისტემა ამოწმებს მომხმარებლის ასაკს (მინ. 18 წელი) და საკრედიტო ქულას.
* **გრაფიკის გენერაცია:** დამტკიცების შემთხვევაში, ავტომატურად ითვლის ყოველთვიურ შესატანს (Annuity ფორმულით) და ქმნის გადახდის გრაფიკს.
* **გადახდების მართვა:** სესხზე თანხის შემოტანის აღრიცხვა და სესხის სტატუსის ავტომატური განახლება (მაგ. დახურვა სრულად დაფარვისას).

## პროექტის გაშვების ინსტრუქცია

იმისათვის, რომ პროექტი ლოკალურად გაუშვათ, აუცილებელია გქონდეთ დაყენებული **.NET SDK** და **SQL Server**. ტერმინალში (პროექტის მთავარი ფოლდერიდან) მიყევით ამ 4 ნაბიჯს:

1.იბლიოთეკების გადმოწერა:
    Bash
    dotnet restore

2.მონაცემთა ბაზის მიგრაციის შექმნა:
    Bash
    dotnet ef migrations add InitialCreate

3.მონაცემთა ბაზის გენერაცია/განახლება:
    Bash
    dotnet ef database update

4.პროექტის გაშვება:
    Bash
    dotnet run
